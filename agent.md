# Agent 协作说明

## 任务进度文档

- **单一事实来源**：根目录 [`TASK_PROGRESS.md`](TASK_PROGRESS.md) 用于跟踪与 **PLAN 对齐** 的阶段任务。
- **人类**：可随时打开 `TASK_PROGRESS.md`，在 **「进度日志」** 表格中 **插入新行**（建议最新在上），或勾选/修改上方各周任务前的 `[ ]` / `[x]`。
- **Agent（编码助手）**：在本仓库完成任务或阶段性工作后，应 **同步更新 `TASK_PROGRESS.md`**：
  - 将对应条目标为 `[x]`，或新增 `[ ]` 子任务；
  - 在 **进度日志** 追加一行：日期 + 简短说明（完成内容、已知问题、后续建议）。
- **不要求** Agent 代替人类填写主观计划；人类插入的日志优先级由人类自行维护。
